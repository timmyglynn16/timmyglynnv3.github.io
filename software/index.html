DEVELOPING ENVIRONMENTS

Visual Studio

Visual Studio is hands down my favorite IDE to work with. I appreciate its simplicity and ability to compile almost every language possible. I love VS's feature to link with my GitHub account and allow me to source control all of my programs. I have over a year and half's worth of experience with Visual Studio for langauges like C++, Java, Python, and web design.

Eclipse

Eclipse was the first IDE I used once I learned what an IDE actually was. I started off using Dr. Java for my first programming class until I discovered Eclipse. It was a lot of fun using dark themes and being able to compile my Java programs a lot easier. Whie I prefer to use VS, Eclipse will always be my first love when it comes to IDE's.

Xcode

I used to be a big Mac guy. I loved everything and anything Mac. When I discovered Xcode on MacOS, I used it for all of my C++ programs. I was still hooked on Eclipse for Java programming, but Xcode was a nice change up for C++ programming. Although it is not my favorite IDE as it is limited to one OS, I love to work with Xcode whenever I get the chance!

Text Editor

Ahh the classic text editor. What's more satisfying than scripting any program in any language in a text editor and running/compiling through the terminal? Nothing. If I am not building an elaborate project and just want to perform some simple tasks or build small algorithms, text editor/terminal is my go-to! I love working in UNIX environments like Ubuntu, so this method of programming is very fun and easy to me!

OPERATING SYSTEMS

macOS

I've always preferred MacOS over Windows. Like I said before, I love everything Apple: iPhones, Macbooks, iPads, etc. I love using Apple machines. macOS is very clean and easy to use. And the best part? It's Linux based so I still get to use my beloved Terminal. I've used macOS my whole life and I'll always feel most comfortable and functional working with it.

Ubuntu

Ubuntu is hands down my go-to operating system today. I love everything about it. It's UNIX based and it's amazing for programming. I hav been using Ubuntu for almost 2 years and I don't plan on changing!

Windows

While I rarely use Windows, I appreciate its applications and user functionailty. Like every other kid from the 90's, I am an expert in Microsoft Office tools and I am extremely familiar with the Windows environment. I mean, who hasn't played over 1,000 games of pinball during computer class in gradeschool? Windows is the classic OS and I always enjoy using it for certain applications when its needed.

SOFTWARE TOOLS

Matlab

MatLab is actually the first time I was ever exposed to computer programming. I was taking a MatLab course in biomedical engineering at Duquesne, and I fell in love with the idea of programming. During that time, I wrote many functions and scripts for mathematical analysis and basic algorithmic operations. For my final project, I simulated scanning for and zapping a blood clot in a human body until the clot reached a desired size.

Wireshark

Network analysis is a big hobby and interest of mine. I discovered network analysis and security in a documentary on HBO and I was hooked. I asked a co-worker at the time how I can get involved in network analysis and he told me about Wireshark. Since then, I've been learning the in's and out's of Wireshark and what it's all about. It's a very powerful and fun tool to work with and there is so much to learn from working with it.

GitHub

I put my entire software life on GitHub. Everything I have ever written/programmed can be seen on my GitHub account. I like using Visual Studio to source control my GitHub repositories and cleanly organize all of my programs.

Arduino

Arduino is a tool I learned in BME and I have been using as a hobby ever since. Programming hardware? Sign me up. I built a laser trip wire for my dorm room, simulated morse code messages using LED's, and tested multiple circuit boards. Arduino is a lot of fun and a very fun tool to work with
